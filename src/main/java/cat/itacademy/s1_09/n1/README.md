# N1 - Enums

## 📌 Exercise Description
Introduction to Java enums: creating type-safe constants, adding behaviour with methods, using enums in classes and handling invalid enum conversions.

## ✨ Exercises
- **Ex1:** Day enum with isWeekend() and isWorkday() methods — prints whether each day is a workday or weekend
- **Ex2:** Level enum (LOW, MEDIUM, HIGH) used in Task class — behaviour changes based on level
- **Ex3:** Level enum with getColor() and getDescription() methods demonstrating enum logic
- **Ex4:** Level.valueOf() converts Strings to enum values — handles invalid values with IllegalArgumentException

## 🛠 Technologies
- Java 21
- Java Enums with methods
- Switch expressions
- Maven
