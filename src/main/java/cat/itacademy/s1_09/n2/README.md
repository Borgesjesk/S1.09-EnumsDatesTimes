# N2 - Dates and Times

## 📌 Exercise Description
Working with the java.time API for robust date and time handling: displaying, calculating differences, arithmetic, formatting, comparing and building an agenda.

## ✨ Exercises
- **Ex1:** Display current LocalDate, LocalTime and LocalDateTime
- **Ex2:** Calculate difference between two dates using Period (years/months/days) and Duration (hours/minutes)
- **Ex3:** Add and subtract days, months, years and hours using plusX() and minusX() methods
- **Ex4:** Format dates with DateTimeFormatter using patterns: dd/MM/yyyy, dd-MM-yyyy, MMMM dd yyyy, EEE MMM dd yyyy
- **Ex5:** isBeforeToday() method comparing a date to LocalDate.now()
- **Ex6:** Agenda of Appointment objects filtered to show only upcoming events, sorted chronologically

## 🛠 Technologies
- Java 21
- LocalDate, LocalTime, LocalDateTime
- Period, Duration
- DateTimeFormatter
- Maven
