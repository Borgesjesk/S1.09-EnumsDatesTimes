# S1.09 - Enums, Dates and Times

## 📌 Exercise Description
Working with Java enums for type-safe constants with behaviour, and the java.time API for robust date and time handling including formatting, arithmetic and comparisons.

## ✨ Functionalities

### Level 1 - Enums
- **Ex1:** Day enum with isWeekend() and isWorkday() methods
- **Ex2:** Level enum (LOW, MEDIUM, HIGH) used in a Task class
- **Ex3:** Level enum with getColor() and getDescription() methods
- **Ex4:** Convert String to enum with valueOf() and handle invalid values

### Level 2 - Dates and Times
- **Ex1:** Display current date, time and datetime with LocalDate, LocalTime, LocalDateTime
- **Ex2:** Calculate difference between dates with Period and Duration
- **Ex3:** Add and subtract days, months, years and hours from dates
- **Ex4:** Format dates with DateTimeFormatter in multiple formats
- **Ex5:** Check if a date is before today
- **Ex6:** Agenda with upcoming appointments sorted chronologically

## 🛠 Technologies
- Java 21
- Java Enums (with methods)
- java.time API (LocalDate, LocalTime, LocalDateTime, Period, Duration, DateTimeFormatter)
- Maven 3.9.11
